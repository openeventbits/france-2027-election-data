# Staging

This directory is for parsed data that has not yet been promoted into the canonical public datasets in `/data`.

Automation should write candidate rows here first, not directly into `/data`.

Staged rows may come from:

- official institution pages
- candidate or party websites
- RSS feeds
- poll notices
- official documents
- geocoding scripts

Rows in staging may be incomplete, duplicated, uncertain, or waiting for review.

Only validated and approved rows should be promoted into `/data`.
