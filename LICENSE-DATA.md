## Attribution

If you reuse the data, please attribute:

FR27 Open Data / Open Event Bits

A suggested citation format is:

FR27 Open Data. Open, versioned, source-linked data for the French 2027 presidential election. Open Event Bits, 2026.

## Source Links

This project links to external public sources. External articles, official pages, documents, poll notices, and media materials remain governed by their original publishers' terms.

FR27 Open Data does not claim ownership over external source material.

## Poll Data

Poll data should be handled carefully.

The project should prioritize poll metadata, public notices, source links, and methodological context. Candidate-level poll values should only be included when the source and reuse context are clear.
